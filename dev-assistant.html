Dev Assistant

on hold
Started Feb 2026 · theaiethos.com/dev

AI-powered mobile development tool. Chat interface for managing the codebase from anywhere — read files, make edits, search code, view history, and create PRs, all powered by Claude with GitHub API tools.

Thesis

Solo developers need to stay productive on the go. An AI assistant with direct GitHub access enables meaningful code work from mobile without a laptop.

What It Does

Token-authenticated chat interface where Claude has full GitHub integration: read, write, search, commits, branches, PRs. Streaming responses for real-time feedback. File tree sidebar, voice input, conversation persistence. Built for iPhone-first mobile development workflow.

Signals

  • Live at theaiethos.com/dev with token auth
  • Full GitHub integration: read, write, search, commits, branches, PRs
  • Streaming responses for real-time feedback
  • File tree sidebar, voice input, conversation persistence

Status

On hold. Functional and deployed as internal tool. Used for quick code changes and PR management from mobile. Not pursuing as an external product — Claude Code and similar tools are solving this space with larger teams.
