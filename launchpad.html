Launchpad

on hold
Started 08 Feb 2026 · launchpad.theaiethos.com

Turn app ideas into prototypes in minutes. AI-powered rapid app scaffolding — describe your idea and Launchpad generates a complete prototype with landing page, auth, feedback widget, onboarding, and docs.

Thesis

Most app ideas die in planning. Launchpad removes friction by auto-generating infrastructure so you can validate faster. Ship in minutes, not months.

What It Does

Describe an app idea in natural language → AI generates a complete proposal (name, landing page copy, feature list, infrastructure config) → edit in a tabbed proposal UI → hit build → streaming real-time progress as each component deploys.

Each app gets its own subdomain via wildcard routing. Auto-configures: Clerk auth, feedback widgets, onboarding flows, and documentation.

Signals

  • AI generates proposals from natural language descriptions
  • Editable proposal UI: customize name, branding, features before build
  • Streaming build progress with real-time SSE status updates
  • Wildcard subdomain routing: each app gets its own URL

Technical Decisions

Dark theme UI to differentiate from Guildry. Each app gets its own CSS file to prevent style bleeding. clerkMiddleware() wrapper for all routes with subdomain routing in Next.js middleware. Fixed global CSS backgrounds overriding nested layouts.

Status

On hold. UI fully built and deployed. Architecture cleanup complete. Would resume as internal tooling for the incubator — each weekly build could be scaffolded through Launchpad.
